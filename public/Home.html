Python

A popular programming language

Learn Python Python Reference
Python Example: 
def calculate_sum(numbers):
    total = 0
    for num in numbers:
        if num > 0:
            total += num
    return total

# Example usage
numbers = [1, -2, 3, 4, -5]
result = calculate_sum(numbers)
print(f"Sum of positive numbers: {result}")
